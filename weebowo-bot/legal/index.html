🔒 Privacy Policy

Last updated: May 04, 2025

1. Information We Collect

WeebOwO may collect and process the following types of data to provide its services:

  • User Data: Discord User IDs, usernames, display names, and avatar URLs
  • Server Data: Guild IDs, channel IDs, role IDs, and server configuration settings
  • Message Data: Message content for moderation purposes, command usage, and bot interactions
  • Usage Data: Command usage statistics, feature interactions, and error logs
  • Persistent Data: User preferences (birthdays, friend codes, AFK status, warnings)

2. How We Use Your Information

Your data is used exclusively to provide WeebOwO's functionality:

  • Processing commands and providing bot responses
  • Server moderation (AutoMod, warnings, logging)
  • Storing user preferences (birthdays, Mario Kart friend codes, AFK messages)
  • Maintaining server configurations (starboard, streaming notifications, counting)
  • Providing music playback and queue management
  • Generating usage statistics for bot improvement

We do not: Sell, rent, or share your data with third parties for commercial purposes.

3. Data Storage & Security

  • Local Storage: Data is stored locally on secure servers
  • Temporary Data: Message content and command data may be cached temporarily for processing
  • Persistent Data: User preferences and server configurations are stored in JSON files
  • Security: We implement reasonable security measures to protect your data
  • Retention: Data is retained only as long as necessary for bot functionality

4. Third-Party Services

WeebOwO integrates with the following external services:

  • Discord API: For all bot functionality and interactions
  • YouTube API: For music playback and video downloads
  • Spotify API: For music streaming integration
  • Tenor API: For GIF responses in interaction commands
  • Trivia API: For trivia game questions
  • Twitch/YouTube APIs: For streaming notifications

These services have their own privacy policies and terms of service.

5. Your Rights

  • Access: Request information about data we have collected about you
  • Deletion: Request deletion of your personal data
  • Correction: Request correction of inaccurate data
  • Opt-out: Stop using the bot to prevent further data collection

To exercise these rights, contact us via Discord or GitHub.

6. Changes to This Policy

We may update this Privacy Policy from time to time. Changes will be posted on this page with an updated "Last updated" date. Continued use of WeebOwO after changes constitutes acceptance of the new policy.
